# Simple Gotcha Website

Structure simple et facile à modifier pour débutant:

- `index.html` — page d'accueil
- `pages/presentation.html` — page Présentation (modifiable)
- `pages/rubrique1.html` — rubrique 1
- `pages/rubrique2.html` — rubrique 2
- `css/styles.css` — styles principaux (changez les couleurs ici)
- `img/` — placez votre logo et images ici

Changer la couleur principale:

1. Ouvrez `css/styles.css`.
2. Modifiez la valeur de `--primary` dans `:root` (remplacez le hex par la couleur de votre logo).

Ajouter votre logo:

1. Remplacez `img/logo-placeholder.svg` par votre logo ou copiez votre logo dans `img/` et modifiez la balise `<img>` dans `index.html`.

Éditer une rubrique:

1. Ouvrez `pages/rubrique1.html` ou `pages/rubrique2.html`.
2. Remplacez le texte, ajoutez des images (placez-les dans `img/`), et sauvegardez.

Visitez le site localement: ouvrez `index.html` dans votre navigateur.
